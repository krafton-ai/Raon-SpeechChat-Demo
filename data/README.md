## raon.wav

Default speaker reference audio used for voice conditioning.

- **Format:** WAV, 16-bit PCM, mono, 44.1 kHz (resampled to 24 kHz at load time)
- **Source:** Provided by KRAFTON as part of the Raon SpeechChat Demo
